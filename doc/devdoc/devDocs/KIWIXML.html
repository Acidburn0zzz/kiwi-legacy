KIWI XML processor and data store The XML object processes the configuration file and stores that data in the imageConfig hash. The hash is layed out as a data structure that resembles the XML file and allows easy access to the data for the use by other objects. Other objects never get access to the internal structure and can only manipulate data via set methods or retrieve data via get methods. Wherever access to complex information is required the XML object will provide a specific object for this data.
