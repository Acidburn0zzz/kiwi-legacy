relocateCatalog

mkisofs/genisoimage leave one sector empty (or fill it with version info if the ISODEBUG environment variable is set) before starting the path table. We use this space to move the boot catalog there. It's important that the boot catalog is at the beginning of the media to be able to boot on any machine
